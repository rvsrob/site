Privacy Poilcy


By inviting & using Thunderstruck into your server, you agree to the following privacy policy!


What type of data do we collect?


 There are a lot of commands in Thunderstruck that can run normally without the needing of databases, but there are categories from Thunderstruck that requires databases, and in those databases we store various types of data. This is all done by using a secured sqlite database!


Each Guild Thunderstruck joins stores the guilds data.

This includes:

Guild ID & Custom Prefix.


Some of Thunderstruck Moderation, Administration & Developer commands will also store users and guild data.

This inlcudes:

User ID, Guild ID and Channel ID.


Why do Thunderstruck need the data?


Thunderstruck needs the data from each guild in order to remember each and every guild's prefixes and more. For example, you set a custom prefix with a custom command in your server, the bot will automatically add the server ID and all the required info it needs to respond on the prefix to the database.


How safe is your data?


The data that's being stored is not open for the public! Its only accessible to Thunderstruck and the bot Owner and Developer Thunder!


How can users contact us regarding their concerns?


In Case of any concerns or questions the users can contact us though support by joining our support guild or by contacting our developer directly by messaging Thunder#0006 on discord!
